This project was completed as part of the application process for the Los Angeles Chargers Football Analysis Fellowship. 
The prompt was as follows: Using whichever public data source(s) you deem appropriate, design and build a new EPA model that you would be confident using to make recommendations to the Head Coach.
All code utilized can be found in the markov_epa_model.R file.
The prompt also asked for two written reports: one intended for the Director of Analytics explaining the modeling choices and innovations, and one intended for the Head Coach using the model to help make decisions for 2nd & Short play selection.
The first report can be found in the analytics_epa_report.pdf file, and the second can be found in the coaching_epa_report.pdf file.
